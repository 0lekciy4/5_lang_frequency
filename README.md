# Frequency Analysis of Words
The script compiles a list of frequently used words in the text.
The path to the file with the text is passed as an argument when
the file is started.
# Example to run.
```bash
$ python3 lang_frequency.py <path_to_file>
num word count
1   и   10649
2   в   5308
3   не   4404
4   что   3954
5   он   3825
6   на   3383
7   с   3108
8   как   2151
9   я   1959
10   его   1937

```
