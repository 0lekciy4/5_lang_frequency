# Frequency Analysis of Words
The script compiles a list of frequently used words in the text.
The path to the file with the text is passed as an argument when
the file is started.
# Example to run.
```bash
$ python3 lang_frequency.py text.txt
1   ('и', 10137)
2   ('в', 4863)
3   ('не', 4221)
4   ('что', 3652)
5   ('на', 3200)
6   ('с', 3025)
7   ('он', 3017)
8   ('как', 1913)
9   ('его', 1876)
10   ('к', 1778)
```
